## Build the Bureau Pro gate with blurred previews and upgrade CTA

**PRIORITY: P1** | **Type: structural**
**Page**: league-intel.html
**Design doc**: docs/bureau-design.md

PRO-gated tabs (Rivals, Trades, History, Scenario Explorer) need a consistent gating experience. Free users should see a blurred preview of the actual content (not a blank page) with a centered overlay containing the feature description and a "Go Pro" CTA button that navigates to pricing.html. The blur should use CSS backdrop-filter and show enough shape/color to be enticing without being readable.

### Task 1: Implement consistent Pro gate component
**Accept when**: All PRO-gated tabs show a blurred preview of real content (not placeholder) with a centered card overlay that includes: feature name, 1-line benefit description, and a "Go Pro" button linking to pricing.html. The gate checks the user's subscription status from localStorage/JWT. The blur amount is enough to obscure text but preserve layout shapes and colors. Gate component is reusable across all locked tabs.
