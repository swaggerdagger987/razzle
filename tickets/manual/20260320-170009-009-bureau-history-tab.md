**PRIORITY: P1** | **Type: structural**

The History tab is PRO-gated. Delivered by Atlas (elephant historian agent). It chains previous_league_id from the Sleeper API to pull multi-season data and surfaces historical patterns: win streaks, playoff appearances, championship odds trajectory over time, roster turnover rate, and recurring draft tendencies. Atlas provides context that no other tool gives — the long view.
