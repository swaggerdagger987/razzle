## Build the Formula Store marketplace for browsing and installing community formulas

**PRIORITY: P1** | **Type: structural**
**Page**: lab.html
**Design doc**: docs/NORTH_STAR.md

The Formula Store is where users browse, preview, and install community-shared screener formulas. It should feel like a curated marketplace with categories (dynasty rankings, trade value, rookie scouting, aging curves). Each formula card shows: name, author, description, column count, install count, and a "Try It" button that loads the formula into the screener. Installed formulas are saved to localStorage alongside user-created ones.

### Task 1: Implement Formula Store browse and install flow
**Accept when**: A "Formula Store" button in the Lab opens a modal/panel with browsable formula cards organized by category. Each card shows name, description, author, and install count. Clicking "Try It" loads the formula's columns and filters into the screener. Installed formulas appear in the user's saved formulas list. At least 5 seed formulas are pre-loaded (dynasty rankings, rookie watch, buy-low candidates, aging veterans, trade value).
