## Build the Rivals tab with Bones' manager behavioral profiles

**PRIORITY: P1** | **Type: structural**
**Page**: league-intel.html
**Design doc**: docs/bureau-design.md (Rivals Tab + Behavioral Tag Classification sections)

The Rivals tab is PRO-gated. Delivered by Bones (skeleton diplomat). This is the conversion trigger.

**Contents:**
1. Bones agent header: "i've been reading the room. here's what your opponents don't want you to know."
2. Manager Profile Cards: one per rival manager with:
   - Name, record, championship odds
   - Behavioral tag: PANIC SELLER / AGGRESSIVE / HOARDER / PATIENT / STEADY
   - Trade frequency (per year), position bias, panic score, FAAB burn rate
3. Bones' annotation per manager: template-driven insight about their behavior

**Behavioral tag rules (first match wins):**
1. PANIC SELLER: >50% trades within 48h of a loss
2. AGGRESSIVE: top 20% trade frequency + net negative pick accumulation
3. HOARDER: bottom 20% trade frequency + net positive pick accumulation
4. PATIENT: bottom 30% trade frequency, no panic pattern
5. STEADY: default fallback

**Minimum data:** 8+ transactions across 1+ seasons. Below threshold: "not enough data."

**Data source:** Sleeper transactions API (trades, waivers, drops) for current + previous seasons via previous_league_id chaining.

**Pro gate:** Show blurred preview on Overview tab. Clicking "Go Pro" navigates to pricing.html.

### Task 1: Implement Rivals tab
**Accept when**: Pro users see one card per rival manager with behavioral tag, trade stats, and Bones' commentary. Free users see the tab locked with a blurred preview and upgrade CTA. Tags are computed from real Sleeper transaction data.
