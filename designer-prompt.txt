Or use Playwright directly:
```python
from playwright.sync_api import sync_playwright
with sync_playwright() as p:
    browser = p.chromium.launch(headless=True)
    page = browser.new_page(viewport={"width": 1280, "height": 900})
    page.goto("http://localhost:8000/PAGE", wait_until="networkidle", timeout=15000)
    import time; time.sleep(3)
    page.screenshot(path="designer-screenshots/PAGE_NAME.png")
    browser.close()
```

Save screenshots to designer-screenshots/ so the Ship agent can reference them.

### Step 3: Look at the screenshot and judge

Read the screenshot. Check against DESIGN.md and the agent connective tissue design:

**Does it load at all?**
- Does the page show data or is it blank/erroring/infinite loading?
- If it doesn't load, that's a P0 ticket. Write it and stop.

**Agent presence (Layer 1 — free users should see this):**
- Are there agent-voiced loading states in Caveat font? ("checking the injury wire...", "scanning the tape...", "pulling film...")
- Are there agent avatar icons (16px) in column header tooltips?
- Are there agent attribution lines (20px icon + name) under panel titles?
- Are there agent-voiced empty states when no data matches?
- Does the 404 page use Razzle's voice?

**Brand compliance (DESIGN.md):**
- Background: sand #ede0cf, cards #f7efe5
- Accent: terracotta #d97757
- Fonts: Luckiest Guy for headings ONLY, Space Mono for ALL data, Caveat for annotations/personality
- Borders: 3px solid, 4px offset shadows on cards
- Position colors: QB blue, RB teal, WR terracotta, TE purple
- No gradients, no thin 1px borders, no cold grays
- Dark mode: espresso flip palette

**Navigation:**
- Nav says "Fourth Down Lab" not "Lab" or "The Lab"
- Nav says "Bureau of Intelligence" not "Bureau" or "League Intel"
- Nav says "Situation Room" not "AI Agents"

**Content/UX:**
- Does this panel show useful data?
- Would a dynasty manager screenshot this?
- Is anything confusing without explanation?
- Are loading states personality-driven or generic "Loading..."?

### Step 4: Find ONE thing wrong

Not ten things. ONE. The smallest, most specific fix you can identify. Examples of good tickets:

- "The breakout panel title says 'Breakouts' with no agent attribution. Add '— Hawkeye' subtitle in Caveat font below the title."
- "The loading state on the trade values panel says 'Loading...' — change to 'reading the room...' in Caveat font (Bones' territory)."
- "The nav link says 'Bureau' — change to 'Bureau of Intelligence'."
- "The injury column in the screener has no agent icon in the tooltip. Add 16px dolphin.svg icon."
- "Panel X shows a blank white area. The API call to /api/endpoint is returning an error. Fix the endpoint or the fetch URL."

Examples of BAD tickets:
- "Make the site feel more like Razzle" — too vague
- "Fix all the panels" — too big
- "Improve the design" — not actionable
