# RETIRED — V2 Cofounder Loop

The three-equals cofounder loop (Opus / Codex / Composer with Gemini board)
ran from cycle 1 to cycle 56. It produced the V2 monorepo.

The ethos has been distilled into:

- `docs/company/OPERATING_SYSTEM.md` (Ethos: three equals on the build chain,
  Karpathy rules, compound intelligence, commit gate, joy of a finished product,
  never stop)
- `docs/company/AUTOMATION.md` (Standard Company Loop, model routing, prompt
  template)
- `docs/company/MEETINGS.md` (Daily Build Standup with three-equals voting,
  Build Review, Founder Board with KEEP/DELETE/REFINE election)

The runner, prompts, and meta-docs live here pending deletion.
