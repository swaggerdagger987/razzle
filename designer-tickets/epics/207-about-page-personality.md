# About page has zero comic-strip personality

## What's wrong

The About page (`about.html`) is five stacked text cards with dashed borders: "Where does the data come from?", "What Razzle is", "Privacy", "Technology", "Contact". No tiger mascot illustration. No Caveat handwritten annotations. No playful asides. No visual personality whatsoever.

DESIGN.md says: "Personality that leaks through the seams: section annotations, player card scribbles, agent editorial asides." The About page — where you explain who Razzle IS — has the least personality of any page on the site.

## Where

`frontend/about.html` — the entire page content.

## Evidence

About page screenshot: monotonous stack of text boxes. No illustrations, no mascot, no handwritten Caveat annotations, no position colors, no visual variety. Compare to the home page which has the tiger emoji, demo briefing cards, pricing badges, and discovery chips.

## Suggested fix

1. Add the tiger mascot illustration (or a large tiger emoji section) at the top — this is Razzle introducing himself
2. Add Caveat handwritten annotations in the margins (e.g., "yep, all free" next to the data section, "no vibes, just numbers" next to technology)
3. Add visual variety: alternate card layouts, a timeline of milestones, agent bio mini-cards
4. The "What Razzle is" section should FEEL like Razzle wrote it — confident, slightly smug, direct

## Why this matters

The About page is where curious users go to evaluate trust. A wall of plain text says "template website." Personality says "real people built this and they care." The About page should be the most Razzle-feeling page on the site, not the least.
