# DQ-032: Agent names on agents.html don't match DESIGN.md

**Priority**: P1 — brand identity
**Page**: agents.html
**Category**: Brand / copy

## Problem

The agent name chips in the hero section (lines 1619-1624) show:
- RAZZLE (matches)
- HAWKEYE (matches)
- **BONES** (DESIGN.md says "The Fox")
- **OCTO** (DESIGN.md says "The Octopus")
- DR. DOLPHIN (matches)
- **ATLAS** (DESIGN.md says "The Elephant")

Three of six agent names don't match the canonical names in DESIGN.md.

## Evidence

- agents.html lines 1619-1624: chip text reads BONES, OCTO, ATLAS
- DESIGN.md line 214: "The Fox (Diplomat)", line 215: "The Octopus (Quant)", line 216: "The Elephant (Historian)"

## Fix

Either:
- **Option A**: Update agents.html chips + all agent references to use DESIGN.md names (The Fox, The Octopus, The Elephant)
- **Option B**: Update DESIGN.md to reflect the renamed agents (Bones, Octo, Atlas) — requires product decision

Recommend Option A unless the rename was intentional and undocumented.
